# App for my students
To jest ordynarna aplikacja terminalowa napisana w C, która z plików txt, w których są ponumerowane pytania z pewnego podręcznika, losuje i wyświetla cztery losowe pytania.

## Cele aplikacji
1. Nauka pisania aplikacji w C oraz kompilowania
2. Stworzenie przydatnego narzędzia do pracy w szkole
